# El Juego de la Vida

## Descripción

**El Juego de la Vida** es una implementación interactiva del famoso **Juego de la Vida de Conway**. Utiliza el **Patrón Prototype** para gestionar las celdas de la cuadrícula, lo que permite clonar y manipular eficientemente el estado de las celdas.

## Características

* **Interactividad:** Alterna el estado de las celdas (viva/muerta) al hacer clic.
* **Simulación:** Pausa, reanuda y ajusta la velocidad de la simulación.
* **Almacenamiento Local:** Guarda y carga partidas usando `localStorage`.
* **Selección y Pegado de Patrones:** Captura áreas de la cuadrícula y pégalos en otras ubicaciones.

## Requisitos

* Navegador moderno que soporte HTML5 y JavaScript.

## Instalación

1. Clona el repositorio:

   ```bash
   git clone https://github.com/tuusuario/TNO-Juego-de-la-Vida.git
   ```

2. Abre `index.html` en tu navegador.

## Uso

* Haz clic en las celdas para alternarlas.
* Usa `Shift + clic y arrastrar` para capturar patrones y luego pégalos en cualquier lugar.
* Controla la simulación y ajusta la velocidad desde la interfaz.
